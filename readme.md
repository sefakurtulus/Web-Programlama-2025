# Spor Salonu Yönetim ve Randevu Sistemi

Bu proje, **Web Programlama** dersi kapsamında geliştirilmiş, ASP.NET Core MVC tabanlı bir Spor Salonu Yönetim ve Randevu Sistemi'dir. Sistemin temel amacı, spor salonu üyelerinin ve yöneticilerinin işlemlerini dijital bir platformda birleştirmektir.

Uygulama, üyelerin online olarak antrenörleri ve sundukları hizmetleri (fitness, yoga, pilates vb.) görüntülemesine, antrenörlerin müsaitlik takvimlerine göre randevu almasına olanak tanır. Yöneticiler için ise salon, hizmet, antrenör ve üye yönetimi gibi işlemleri gerçekleştirebilecekleri bir admin paneli bulunmaktadır. Projenin öne çıkan özelliklerinden biri, kullanıcıların vücut bilgilerini girerek kişiselleştirilmiş egzersiz ve diyet önerileri alabileceği **OpenAI API entegrasyonudur**.

**Teknolojiler:** ASP.NET Core MVC, Entity Framework Core, SQL Server/PostgreSQL, REST API, Bootstrap 5, C#.
